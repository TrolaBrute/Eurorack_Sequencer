# Eurorack_Sequencer

This is an effort to take a first step into eurorack module manufuacturing. A sequencer was chosen as the first module because theres no need for careful audio signal maintenance, instead
focusing on producing stable control voltages at the output. This allows room for learning integration with the standard eurorack spec, as well as PCB development proccess and microcontroller
coding.

Included will be all files necessary to recreate the project, including PCB files, CAD Files, BOM, and Arduino Sketch
